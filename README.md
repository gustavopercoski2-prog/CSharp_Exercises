## 💻 Repositório para aprendizagem de C#

Este repositório é o meu portfólio de estudos em C# e contém todos os exercícios, projetos e anotações desenvolvidos durante o curso **[C# COMPLETO Programação Orientada a Objetos + Projetos](https://www.udemy.com/course/programacao-orientada-a-objetos-csharp/)**, ministrado pelo professor **Nelio Alves** na plataforma **Udemy**.

## 🎯 Objetivo
Documentar minha evolução prática no aprendizado da linguagem C#, framework .NET e Programação Orientada a Objetos (POO), aplicando boas práticas de desenvolvimento, Clean Code e versionamento de código.

## 📚 O que já aprendi e apliquei até agora

Durante a resolução dos exercícios, coloquei em prática diversos conceitos básicos de programação em C#. Trabalhei com entrada e saída de dados no console, estruturas condicionais (`if`, `else if`, `else`) para controle de fluxo e loops como `while` e `for` para repetições. Também utilizei lógica algorítmica com contadores, validação de dados, cálculo de médias e conversão de tipos.

### 📦 Programação Orientada a Objetos (Introdução)

Também comecei a aplicar conceitos iniciais de Programação Orientada a Objetos, criando classes como `Pessoa` e `Funcionario`, organizando o projeto com uma classe por arquivo (`.cs`) e utilizando `public` para permitir a comunicação entre o `Program.cs` e as demais classes.

## 📁 Estrutura do repositório (resumo)

- `SectionXX_*` → Cada seção do curso tem sua própria pasta
- `COMANDOS.md` → Anotações rápidas de comandos úteis
- `.sln` → Solução compatível com VS Code / Visual Studio

## 🐧 Ambiente de desenvolvimento

Estou desenvolvendo este projeto no Linux (Ubuntu), utilizando o .NET SDK 8.0 e o Visual Studio Code.

### Instalação do .NET SDK 8.0 no Ubuntu

Caso ainda não tenha instalado, execute no terminal:

```bash
sudo apt-get update
sudo apt-get install -y dotnet-sdk-8.0
```
